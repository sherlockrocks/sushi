# Sush — GitHub Pages Sample Website

This repository contains a simple animated website designed for GitHub Pages.

## Pages included

- `index.html` — homepage with hero animation and interactive details
- `about.html` — story page with values and timeline
- `contact.html` — contact form with motion and form feedback

## How to deploy

1. Push this repository to GitHub.
2. In the repository settings, enable GitHub Pages from the `main` branch (or your default branch).
3. Choose the root directory as the publishing source.

## Quick commands

```bash
cd path/to/sushi.github.io
git add .
git commit -m "Add Amora GitHub Pages website"
git push origin main
```

> If your branch is named `master`, replace `main` with `master`.

## Notes

- The contact form is static and uses JavaScript for demo feedback only.
- The site is fully static, so it works perfectly on GitHub Pages.
